Помощь по TenderTracker
Как использовать приложение:
  • Выберите тип закупок - отметьте нужные категории для поиска
  • Настройте регионы - выберите федеральные округа для поиска
  • Запустите поиск - нажмите кнопку "Найти закупки"
  • Скачайте результат - после завершения поиска скачайте Excel файл
Подсказки:
  • Поиск может занять несколько минут в зависимости от количества найденных закупок
  • Для точного поиска выбирайте конкретные регионы
  • Результаты сохраняются в формате Excel для удобного анализа
